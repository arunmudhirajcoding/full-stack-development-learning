Types of Selectors in CSS

  • Tag Selector (e.g. p)
  • ID Selector (e.g. #article1)
  • Class Selector (e.g. .paraClass)
  • Universal Selector (e.g. *)
  • Attribute Selector (e.g. [type="text"])
  • Combinator Selector (e.g. p.paraClass)
  • Pseudo-class Selector (e.g. :hover)
  • Pseudo-element Selector (e.g. ::before)

Types of Combinator Selectors in CSS

  • Child Combinator (e.g. p > span)
  • Descendant Combinator (e.g. p span)
  • Adjacent Sibling Combinator (e.g. p + span)
  • General Sibling Combinator (e.g. p ~ span)
  • Attribute Selectors: *= (contains) ^= (starts with) $= (ends with)
  • Examples:
    • input[type="text"]
    • input[name^="user"]
    • input[name$="name"]
